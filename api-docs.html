API Reference

Comprehensive documentation for all API endpoints

POST /v1/remove-background Primary

Remove Background

Main endpoint for background removal. Supports multiple image formats and providers.

Request Parameters

image_file
Required. Image file to process (JPG, PNG, WebP, max 10MB)
size
Optional. Output size: 'auto', 'preview', 'full' (default: 'auto')
format
Optional. Output format: 'png', 'jpg', 'webp' (default: 'png')
provider
Optional. Force specific provider: 'removebg', 'photoroom' (default: auto-select)

Response

{
  "success": true,
  "image_url": "https://api.clearbg.com/results/abc123.png",
  "provider_used": "removebg",
  "credits_used": 1,
  "processing_time": 2.3
}

Example Request

curl -X POST https://api.clearbg.com/v1/remove-background \
  -H "X-API-Key: YOUR_API_KEY" \
  -F "image_file=@/path/to/image.jpg" \
  -F "size=full" \
  -F "format=png"

JavaScript SDK

Simple integration for web applications with automatic fallback and retry logic.

Installation

// NPM
npm install clearbg-js

// CDN
<script src="https://cdn.clearbg.com/js/clearbg.min.js"></script>

Basic Usage

import ClearBG from 'clearbg-js';

const clearbg = new ClearBG({
  apiKey: 'your-api-key',
  fallback: true // Enable automatic fallback
});

// Remove background from file input
async function removeBackground(file) {
  try {
    const result = await clearbg.removeBackground({
      imageFile: file,
      size: 'full',
      format: 'png'
    });
    
    console.log('Result:', result.imageUrl);
    return result.imageUrl;
  } catch (error) {
    console.error('Processing failed:', error);
  }
}

Advanced Features

Batch Processing

const batch = await clearbg.batchRemove({
  images: [file1, file2, file3],
  webhook: 'https://your-app.com/webhook'
});

console.log('Batch ID:', batch.batchId);

Progress Tracking

const result = await clearbg.removeBackground({
  imageFile: file,
  onProgress: (progress) => {
    console.log(`Progress: ${progress}%`);
  }
});

Provider Selection

// Force specific provider
const result = await clearbg.removeBackground({
  imageFile: file,
  provider: 'removebg' // or 'photoroom'
});
